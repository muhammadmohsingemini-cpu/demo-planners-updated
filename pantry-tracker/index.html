Best Practices

Use expiry dates. Approximate dates still make alerts useful.
Set minimum stock. Staples become grocery suggestions before they run out.
Bulk add first. A kitchen audit is faster when pasted line by line.
Export backups. Browser data is private, so backups protect it.

FAQ

Where is my data stored?

In the full version, data is saved in this browser using localStorage.

Can I create custom storage areas?

Yes. Setup supports default and custom areas, and custom areas appear under Other.

How are grocery suggestions made?

Expired and low-stock inventory items become suggestions unless they are already on your active list.

Can I move data to another browser?

Export a JSON backup, then import it in the other browser.
